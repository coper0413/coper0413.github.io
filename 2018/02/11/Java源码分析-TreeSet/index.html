发表于 2018-02-11
TreeSet

TreeSet实现了SortedSet接口，它是一个有序的集合类，TreeSet的底层是通过TreeMap实现的。TreeSet并不是根据插入的顺序来排序，而是根据实际的值的大小来排序。TreeSet支持两种排序方式：自然排序、自定义排序。
